# odin-recipies
This is the first project in The Odin Project course.

In this project I have been tasked with creating a simple website with a homepage and three other pages each
containing their own recipe.
The index links to each page while each page links back to the index.
Each recipe page must have a header with the recipe name, an image of the recipe, a description, ingredients and
finally a steps section.

Essential elements for html page creation:
<html>, <head>, <body>, 

Within the head I also used:
<meta> and <title>

In this project I gained experience with the basic creation of an html page and the common elements that are used within it.
These elements include: 
header<h1><h2>, paragraph<p>, image<img>, ordered list<ol>, unordered list<ul>, list<li>, 
anchor<a>, bold<strong>, italicize<em>

I have also gained much needed experience using git to add, commit and push in order to keep versions of my
project accessible in the repository
